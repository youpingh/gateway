-- Dev Notes:
Nothing fancy, simple HTML/JavaScript
Uses OpenAI's translate API to translate the blog from Chinese to English. The translated results are saved in the browser's LocalStorage, so if you read the blogs in English in one device, it translates the blog only once unless you do 're-translate'.
