Learners API Gateway

This server provides the following backend services:

  • Text-to-Speech (TTS): POST /tts
  • Speech-to-Text (STT): POST /stt
  • Pinyin Conversion: POST /pinyin
  • Translation: POST /translate

Each endpoint expects a JSON payload with the necessary parameters.

Example request (for /pinyin):

{
  "text": "你好"
}
